\chapter{Удиртгал}
\label{ch:introduction}

\section{Үндэслэл, ач холбогдол}

Орчин үеийн робот техникийн хөгжил нь бүрэн автомат системүүдийн найдвартай байдал болон бодит орчинд дасан зохицох чадварт өндөр шаардлага тавьж байна. ABU Robocon 2026 тэмцээний дүрмийн дагуу R2 робот нь хүний оролцоогүйгээр судрыг таньж авах, бие даан стратегийн шийдвэрийг гаргах ёстой. Энэ нь объект таних (Object Detection) болон шийдвэр гаргах алгоритмыг нэгтгэж, тэмцээний орчинд тогтвортой ажиллах алгоритмыг шаардаж байна.

Судалгааны онолын ач холбогдол нь YOLOv12 объект таних алгоритм болон Finite State Machine (Төгсгөлөг Төлөвт Машин) шийдвэр гаргах алгоритмыг Unity симуляцийн орчинд нэгтгэж хэрэгжүүлсэн алгоритмыг тодорхойлоход оршино. Энэ нь симуляцийн орчинд баталгаажиж, уг шийдэл нь Шинэ Монгол Технологийн Коллежоос тэмцээнд оролцох New Robo багийн R2 роботын бэлэн суурь болох бөгөөд, цаашид бодит hardware дээр хэрэгжүүлэх боломжийг нээнэ.

\section{Зорилго, зорилт}

\subsection{Судалгааны зорилго}

ABU Robocon 2026 тэмцээнд зориулсан R2 бүрэн автомат роботын объект таних болон шийдвэр гаргах алгоритмын хэрэгжүүлж, Unity симуляцийн орчинд турших болон баталгаажуулах.

\subsection{Судалгааны зорилтууд}

Дээрх зорилгод хүрэхийн тулд дараах зорилтуудыг дэвшүүлсэн:

\begin{enumerate}
    \item ABU Robocon 2026 тэмцээний талбай болон судрын загварыг Unity симуляцийн орчинд бэлтгэх.
    \item YOLOv12 алгоритмыг ашиглан симуляцийн орчноос датасет бэлтгэж, объект таних алгоритмыг сургаж хэрэгжүүлэх.
    \item Finite State Machine алгоритмыг ашиглан R2 роботын шийдвэр гаргах алгоритмыг симуляцийн орчинд хэрэгжүүлэх.
\end{enumerate}
